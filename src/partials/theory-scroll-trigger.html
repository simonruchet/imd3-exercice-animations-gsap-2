Rappel théorique - Scroll Trigger GSAP

Lien vers le support de cours sur le ScrollTrigger GSAP

Qu'est-ce que ScrollTrigger ?

ScrollTrigger est un plugin GSAP qui permet de déclencher des animations en fonction de la position de scroll de la page. Il offre un contrôle précis sur quand et comment les animations se déclenchent lors du défilement.

Installation et utilisation de base

// Enregistrer le plugin
gsap.registerPlugin(ScrollTrigger);

// Animation simple déclenchée au scroll
gsap.to(".box", {
  x: 100,
  duration: 2,
  scrollTrigger: ".box" // Se déclenche quand .box entre dans le viewport
});

Configuration des points de contact (trigger points)

ScrollTrigger utilise des points de déclenchement pour contrôler les animations : 

gsap.to(".box", {
  x: 100,
  duration: 2,
  scrollTrigger: {
    trigger: ".box",        // Élément qui déclenche l'animation
    start: "top bottom",    // Quand le top de .box touche le bottom du viewport
    end: "bottom top",      // Quand le bottom de .box touche le top du viewport
    scrub: true,            // Animation liée au scroll
    markers: true           // Afficher les marqueurs (debug)
  }
});

Options de déclenchement

Points de contact (start/end)

// Format: "trigger viewport"
start: "top bottom"     // Top de l'élément touche le bottom du viewport
start: "center center"  // Centre de l'élément au centre du viewport
start: "bottom top"     // Bottom de l'élément touche le top du viewport

// Avec des offsets
start: "top bottom-=100px"  // 100px avant que le top touche le bottom
