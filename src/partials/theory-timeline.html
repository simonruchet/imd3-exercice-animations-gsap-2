Rappel théorique - Timeline GSAP

Lien vers le support de cours sur les animations GSAP

Qu'est-ce qu'une Timeline ?

Une Timeline GSAP est un conteneur qui permet de séquencer et synchroniser plusieurs animations. Elle offre un contrôle précis du timing et permet de créer des animations complexes avec des délais relatifs entre les éléments.

Créer une Timeline

// Créer une timeline vide
const tl = gsap.timeline();

// Créer une timeline avec des options
const tl = gsap.timeline({
  repeat: 2,        // Répéter 2 fois
  yoyo: true,       // Aller-retour
  delay: 1,         // Délai avant le début
  paused: true      // Commencer en pause
});

Ajouter des animations à une Timeline

Les animations s'ajoutent en séquence par défaut :

const tl = gsap.timeline();

// Les animations se déclenchent l'une après l'autre
tl.to(".box1", { x: 100, duration: 1 })
  .to(".box2", { y: 100, duration: 1 })
  .to(".box3", { rotation: 360, duration: 1 });

Contrôler le timing relatif

Le troisième paramètre permet de contrôler quand l'animation démarre :

Position relative

tl.to(".box1", { x: 100, duration: 1 })
  .to(".box2", { y: 100, duration: 1 }, "-=0.5")  // Démarre 0.5s avant la fin de la précédente
  .to(".box3", { rotation: 360, duration: 1 }, "+=0.2");  // Démarre 0.2s après la fin

Position absolue

tl.to(".box1", { x: 100, duration: 1 })
  .to(".box2", { y: 100, duration: 1 }, 0.5)  // Démarre à 0.5s depuis le début de la timeline
  .to(".box3", { rotation: 360, duration: 1 }, 2);  // Démarre à 2s depuis le début

Labels pour organiser

tl.addLabel("start")
  .to(".box1", { x: 100, duration: 1 })
  .addLabel("middle")
  .to(".box2", { y: 100, duration: 1 })
  .to(".box3", { rotation: 360, duration: 1 }, "middle+=0.5");  // 0.5s après le label "middle"
