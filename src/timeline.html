Exercices - Timeline GSAP

Consigne

Depuis js/timeline.js, écrivez le code JavaScript pour créer des animations avec des timelines GSAP selon les indications de chaque exercice.

Chaque exercice doit utiliser une timeline pour séquencer les animations !

Exemple

// Créer une timeline et animer plusieurs éléments en séquence
const tl = gsap.timeline();
tl.to("#js-timeline-1", { x: 100, duration: 1 })
  .to("#js-timeline-1", { y: 50, duration: 0.5 }, "-=0.2"); // Commence 0.2s avant la fin de la précédente
1
2
3
